# AXI4-Lite-UART-SoC-Peripheral-with-FIFO-CDC-and-Interrupt-on-Artix-7-FPGA-
Designed AXI4-Lite UART SoC peripheral with TX/RX FIFO, baud rate generator, CDC and interrupt logic using Verilog HDL. Also Verified using simulation and implemented on Artix-7 FPGA including synthesis, implementation and bitstream generation.
